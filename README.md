## 🛠️ 快速开始

### 环境要求

- Rust 1.70+
- 8GB+ 内存
- 多核CPU (推荐8核以上)

### 安装运行

#### Linux/macOS

```bash
chmod +x run_accelerator.sh
./run_accelerator.sh
```

#### Windows

```cmd
run_accelerator.bat
```

### 手动构建运行

```bash
# 构建项目
cargo build --release

# 运行加速系统
cargo run --release
```
